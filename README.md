# ESP32-S3 Rubber Ducky

Educational demonstration of HID Keystroke Injection attack using ESP32-S3.

## Features
- Auto popups with IP address and "Hacked" message
- Sets system volume to maximum
- Opens YouTube video
- Opens Camera app and attempts to take photo
- Creates "YOU_HAVE_BEEN_HACKED.txt" on Desktop
- Opens the hacked note automatically

## Board Setup (Arduino IDE)
- Board: **ESP32S3 Dev Module**
- USB CDC On Boot: **Enabled**
- USB Mode: **USB-OTG (TinyUSB)**
- Connect to the **Native USB port**

## Warning
This project is for **educational and cybersecurity awareness** purposes only.  
Do not use it for malicious activities.
